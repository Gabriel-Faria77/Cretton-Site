# Cretton Modas — Website Institucional

Aplicação web desenvolvida para a Cretton Modas com o objetivo de apresentar a empresa, exibir produtos e facilitar o contato com clientes por meio de canais digitais.

---

## Visão Geral

O site atua como uma vitrine digital contendo:

* Apresentação institucional
* Exibição de produtos
* Integração com redes sociais
* Canal direto via WhatsApp
* Layout responsivo

---

## Estrutura do Projeto

```
/cretton-site
│
├── index.html
├── css/
│   └── style.css
├── js/
│   └── script.js
├── data/
│   └── empresa.js
└── img/
```

---

## Arquitetura

O projeto segue o princípio de separação de responsabilidades:

* HTML: estrutura
* CSS: estilização
* JavaScript: comportamento
* Data (empresa.js): conteúdo dinâmico

---

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript (Vanilla)

---

## Execução

1. Clonar o repositório:

```
git clone https://github.com/seu-usuario/cretton-site.git
```

2. Acessar a pasta:

```
cd cretton-site
```

3. Abrir o arquivo `index.html` no navegador.

---

## Customização

As informações da empresa estão centralizadas em:

```
data/empresa.js
```

É possível alterar:

* Nome da empresa
* Descrição
* Endereço
* Produtos
* Links de redes sociais
